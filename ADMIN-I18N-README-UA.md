# 🌍 Мультимовна система адмін-панелі

## Огляд

Інтерфейс адміністрування тепер доступний **2 мовами**:
- 🇫🇷 **Французька** (за замовчуванням)
- 🇺🇦 **Українська**

## Функціональність

### Перемикач мови
Перемикач мови доступний у верхньому правому куті інтерфейсу адміністрування:
- Натисніть 🇫🇷 FR для переходу на французьку
- Натисніть 🇺🇦 UA для переходу на українську

### Збереження налаштувань
Обрана мова автоматично зберігається в браузері (localStorage) і буде відновлена при наступному відвідуванні.

### Доступні переклади

Всі елементи інтерфейсу перекладені:
- ✅ Меню навігації (Панель, Контент, Медіа)
- ✅ Заголовки та підзаголовки
- ✅ Кнопки дій
- ✅ Повідомлення-сповіщення
- ✅ Мітки форм
- ✅ Повідомлення про помилки
- ✅ Модальні вікна та спливаючі повідомлення

## Технічна архітектура

### Задіяні файли

1. **`js/admin-i18n.js`** : Модуль перекладу
   - Містить всі переклади (FR + UA)
   - Керує зміною мови
   - Автоматично оновлює інтерфейс

2. **`admin.html`** : Атрибути `data-i18n`
   - Кожен елемент, що перекладається, має атрибут `data-i18n="ключ.перекладу"`
   - Приклад: `<h2 data-i18n="sidebar.title">🏋️‍♀️ Alina Admin</h2>`

3. **`css/admin-styles.css`** : Стилі перемикача
   - Стилі для кнопок мови
   - Позиціонування у верхньому правому куті

### Структура перекладів

```javascript
adminTranslations = {
    fr: {
        sidebar: {
            title: "🏋️‍♀️ Alina Admin",
            dashboard: "Dashboard",
            content: "Contenu",
            // ...
        },
        notifications: {
            contentSaved: "✅ Modifications enregistrées avec succès!",
            // ...
        }
    },
    ua: {
        sidebar: {
            title: "🏋️‍♀️ Alina Адмін",
            dashboard: "Панель",
            content: "Контент",
            // ...
        },
        notifications: {
            contentSaved: "✅ Зміни успішно збережено!",
            // ...
        }
    }
}
```

## Додавання нової мови

Щоб додати нову мову (наприклад, словацьку):

### 1. Додати переклади в `admin-i18n.js`

```javascript
const adminTranslations = {
    fr: { /* ... */ },
    ua: { /* ... */ },
    sk: {  // Нова мова
        sidebar: {
            title: "🏋️‍♀️ Alina Admin",
            dashboard: "Dashboard",
            content: "Obsah",
            media: "Médiá",
            userRole: "Lokálna administrácia"
        },
        // ... скопіювати всю структуру та перекласти
    }
};
```

### 2. Додати кнопку в `admin.html`

```html
<div class="admin-language-selector">
    <button class="admin-lang-btn active" data-lang="fr">🇫🇷 FR</button>
    <button class="admin-lang-btn" data-lang="ua">🇺🇦 UA</button>
    <button class="admin-lang-btn" data-lang="sk">🇸🇰 SK</button> <!-- Нова -->
</div>
```

### 3. Це все!

Система автоматично виявить нову мову та застосує її.

## Використання в коді

### Отримати переклад

```javascript
// Метод 1: Через глобальний екземпляр
const text = adminI18n.t('sidebar.title');

// Метод 2: Допоміжна функція (рекомендовано)
const t = (key) => typeof adminI18n !== 'undefined' ? adminI18n.t(key) : key;
const text = t('sidebar.title');
```

### Змінити мову програмно

```javascript
adminI18n.setLang('ua'); // Перейти на українську
adminI18n.setLang('fr'); // Перейти на французьку
```

### Отримати поточну мову

```javascript
const currentLang = adminI18n.getCurrentLang(); // 'fr' або 'ua'
```

## Найкращі практики

1. **Завжди використовуйте ключі перекладу** замість жорстко закодованого тексту
   ```javascript
   // ❌ Погано
   button.textContent = 'Зберегти';
   
   // ✅ Добре
   button.textContent = t('common.save');
   ```

2. **Додавайте атрибут `data-i18n`** до HTML-елементів
   ```html
   <!-- ❌ Погано -->
   <h2>Заголовок</h2>
   
   <!-- ✅ Добре -->
   <h2 data-i18n="section.title">Заголовок</h2>
   ```

3. **Групуйте переклади за контекстом**
   ```javascript
   {
       sidebar: { /* ... */ },
       header: { /* ... */ },
       notifications: { /* ... */ },
       common: { /* ... */ }
   }
   ```

## Усунення несправностей

### Мова не змінюється
- Перевірте, що `admin-i18n.js` завантажується **перед** іншими скриптами
- Перевірте консоль на наявність помилок JavaScript
- Очистіть кеш браузера

### Деякі тексти не перекладені
- Перевірте наявність атрибута `data-i18n`
- Перевірте, що ключ існує в обох мовах
- Викличте `adminI18n.updateUI()` для примусового оновлення

### Налаштування не зберігаються
- Перевірте, що localStorage увімкнено в браузері
- Перевірте консоль на наявність помилок

## Обслуговування

### Додати новий переклад

1. Відкрити `js/admin-i18n.js`
2. Додати ключ у **всі** мови
3. Використовувати ключ у коді з `t('новий.ключ')`

### Змінити існуючий переклад

1. Відкрити `js/admin-i18n.js`
2. Знайти ключ для зміни
3. Змінити текст у потрібній мові
4. Перезавантажити сторінку адміністрування

## Підтримка

При виникненні питань або проблем:
- Перевірте консоль JavaScript (F12)
- Переконайтеся, що всі файли завантажені
- Протестуйте в іншому браузері
