# Localhost JSON Admin - Інструкції

## Передумови
- Node.js встановлено

## Запуск
З папки `coaching-website`:

```bash
npm start
```

## URL-адреси
- Публічний сайт: http://localhost:3000/index.html
- Адмін-панель: http://localhost:3000/admin.html
- API контенту JSON: http://localhost:3000/api/content

## Збереження даних
Дані зберігаються в:

```text
data/site-content.json
```

## Поточна функціональність
- локальний сервер обслуговує сайт
- сервер надає локальний JSON API
- читання контенту доступне через `GET /api/content`
- збереження контенту доступне через `PUT /api/content`

## Наступний крок
Залишається підключити адмін-панель (`admin.html` / `js/admin.js`) для:
- завантаження JSON з `/api/content`
- редагування розділів
- збереження змін у `data/site-content.json`

## 🌍 Мультимовність адмін-панелі

Інтерфейс адміністрування доступний **2 мовами**:
- 🇫🇷 Французька (за замовчуванням)
- 🇺🇦 Українська

### Як змінити мову:
1. Відкрийте адмін-панель: http://localhost:3000/admin.html
2. Натисніть на кнопку мови у верхньому правому куті:
   - 🇫🇷 FR для французької
   - 🇺🇦 UA для української
3. Інтерфейс миттєво змінюється!

Ваш вибір мови автоматично зберігається в браузері.
